# OMX Product Ultrawork

Use this plugin skill when the repo-local OMX plugin bundle is installed and you want the same default path the root skill catalog exposes.

- Start with `omx session start`
- Create or resume planning artifacts with `omx autoresearch init <phase>` when discovery is needed
- Queue durable work with `omx team queue`
- Keep the runtime honest with `omx hud`
