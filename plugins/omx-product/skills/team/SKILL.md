# OMX Product Team

Use this plugin skill when the OMX plugin bundle is installed and you want durable team mode from inside Codex.

- Initialize the runtime with `omx team init`
- Spawn workers with `omx team spawn <workerId>`
- Route work through queue, claim, complete, and review
- Use `omx team inbox` and `omx team logs <workerId>` to keep the runtime honest
