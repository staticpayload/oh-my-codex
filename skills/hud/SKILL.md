# HUD

Use `omx hud` before assuming the runtime state is correct.

## Modes

- `omx hud`
- `omx hud --json`
- `omx hud --watch`

The HUD should stay aligned with `.omx/` reality. If it says a mode is active, there must be durable evidence.
