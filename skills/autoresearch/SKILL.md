# Autoresearch

Use this when the work benefits from a durable discovery or optimization loop.

## Commands

- `omx autoresearch init <mission>`
- `omx autoresearch status`
- `omx autoresearch log <message>`

Keep the mission explicit and log phase changes so the HUD can reflect them.
